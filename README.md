# AI API Latency Test

A small tool to test AI API latency, streaming TTFT, model listing, and OpenAI-compatible endpoint behavior.

Website: https://apilatencytest.com

## Features

- Test `GET /v1/models`
- Test `POST /v1/chat/completions`
- Measure chat latency and streaming time to first token
- Check OpenAI-compatible response shapes
- Fetch available models from a base URL
- No account and no stored API keys
